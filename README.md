This README is written for Windows users. You do not need coding experience to get started.

2. Find the latest release at the top of the list.

If the release includes more than one file, choose the one for Windows. It may end in `.exe`, `.msi`, or `.zip`.

## 🪟 Install on Windows

### If you downloaded an `.exe` file

1. Double-click the file.
2. If Windows asks for permission, select **Yes**.
3. Follow the setup steps on the screen.
4. Finish the install.

### If you downloaded an `.msi` file

1. Double-click the file.
2. Follow the installer steps.
3. Choose **Next** when asked.
4. Finish the install.

### If you downloaded a `.zip` file

1. Right-click the file.
2. Choose **Extract All**.
3. Pick a folder, such as **Desktop**.
4. Open the extracted folder.
5. Double-click the app file inside the folder.
